# Exploratory Data Analysis (EDA) - DecodeLabs Project 2

## Project Overview
This project represents the completion of **Project 2: Exploratory Data Analysis (EDA)** during the Data Analytics Internship at DecodeLabs (Batch 2026). The objective was to transform a raw dataset into actionable insights by uncovering hidden patterns, trends, and distributions.

## Key Objectives
- **Data Cleaning & Preparation**: Processing raw order data for analysis.
- **Descriptive Statistics**: Calculating mean, median, and counts to understand data behavior.
- **Trend Analysis**: Identifying top-performing products and order status distributions.
- **Insight Generation**: Translating numerical findings into business-relevant observations.

## Project Summary
Based on an analysis of **1,200 orders**, the following key insights were derived:

| Metric | Value |
| :--- | :--- |
| **Average Order Total** | 1,053.97 |
| **Median Order Total** | 823.61 |
| **Most Ordered Product** | Printer |
| **Highest Transaction** | 3,456.40 |

### Key Observations
* **Order Volume**: The dataset comprises 1,200 unique order records.
* **Product Performance**: 'Printer', 'Tablet', and 'Chair' are among the top-ordered items.
* **Distribution**: Discrepancies between the mean and median indicate significant variance in order values, likely due to outliers.
* **Operational Status**: Order statuses are distributed across Cancelled, Returned, Pending, Shipped, and Delivered categories.

## Technologies Used
* **Python**: Used for data manipulation and statistical calculation.
* **Pandas/Excel**: Tools utilized for data processing and analysis.
* **Analytical Techniques**: Univariate analysis, Five-Number Summary, and distribution evaluation.
